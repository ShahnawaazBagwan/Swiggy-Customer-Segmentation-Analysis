# Swiggy-Customer-Segmentation-Analysis

Swiggy-inspired data analytics project, where I analyzed food delivery and campaign performance data to derive actionable business insights.

Project Highlights:

Used SQL for querying, filtering, and aggregating large datasets

Worked on Databricks for scalable data processing and analysis

Utilized Excel to maintain results, perform validations, and generate clear visual dashboards

Analyzed metrics like exposed orders, clicked orders, CTR, and city-wise performance

Identified trends and performance gaps across campaigns and locations

Key Learnings:

How data pipelines support analytics in large-scale platforms like Swiggy

Translating raw data into meaningful insights for business decisions

Importance of data validation and visualization for stakeholders

Tech Stack:

SQL

Databricks

Excel

Data Analysis & Visualization

This project gave me hands-on exposure to real-world data workflows and strengthened my analytics foundation.
